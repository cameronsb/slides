About This Collection

This is a collection of interactive slideshow presentations featuring AI-generated narration using OpenAI's text-to-speech technology. Each presentation is designed to be engaging, informative, and accessible.

Features:

  • High-quality AI narration with adjustable playback speed
  • Keyboard navigation and speaker notes
  • Responsive design for all devices
  • No dependencies - pure HTML/CSS/JavaScript

Navigation: Use arrow keys to navigate, spacebar to play/pause audio, and 'S' to toggle speaker notes.
